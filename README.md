# Kartenblick

Mobile-first MVP für das Erkennen und Einordnen von Pokémon-Karten.

## Lokal starten

```bash
npm install
npm run dev
```

Dann `http://localhost:3000` öffnen.

## Projektstruktur

```text
src/app/              # Next.js-Oberfläche
packages/domain/      # gemeinsame Domänen-Typen
docs/                 # Produkt- und Datenquellenentscheidungen
.github/workflows/    # CI für Lint und Production-Build
```

Der MVP-Plan befindet sich im übergeordneten QuickScan-Ordner in `pokemon-karten-mvp-plan.md`.
